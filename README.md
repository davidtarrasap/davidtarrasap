#### 🤖 AI Engineer Intern · TuringDream
- Desarrollo de flujos de trabajo con **LLMs, agentes y herramientas de IA**

#### 🗺️ GeoAI Tourism Route Recommender
Trabajo de Fin de Grado centrado en un sistema de recomendación de rutas turísticas personalizadas.

- Sistema híbrido de recomendación con señales colaborativas, de contenido y geoespaciales

- Backend con FastAPI
- Visualización interactiva de rutas y mapas
